# 原神 キャラ育成管理

原神のキャラクター育成状況を管理するWebアプリです。ビルド不要で、ブラウザですぐに使えます。

## 機能

- **キャラ登録・管理** — 70体以上のキャラから選択して登録
- **育成状況トラッキング** — レベル、命ノ星、天賦、武器、聖遺物
- **ダッシュボード** — 優先育成キャラ、進捗サマリー
- **素材計算** — レベルアップ・突破・天賦に必要な素材を概算
- **フィルター** — 元素・武器種・レアリティ・名前検索
- **ローカル保存** — データはブラウザに自動保存（サーバー不要）

## 使い方

1. `index.html` をブラウザで開く
2. 「キャラ一覧」→「+ キャラ追加」からキャラを登録
3. カードをクリックして詳細を編集・保存
4. 「素材計算」で必要素材を確認

### ローカルサーバーで開く（推奨）

ES Modules を使っているため、一部ブラウザでは `file://` ではなくローカルサーバー経由が安定します。

Python が入っている場合:

```bash
cd genshin-builder
python -m http.server 8080
```

ブラウザで https://localhost:8080 を開いてください。

## ファイル構成

```
genshin-builder/
├── index.html          # メインHTML
├── css/style.css       # スタイル
├── js/
│   ├── app.js          # アプリ本体
│   ├── storage.js      # ローカルストレージ
│   └── data/
│       ├── characters.js  # キャラデータ
│       └── materials.js   # 素材計算
└── README.md
```
